ประวัติ

จังหวัดสมุทรสาครเดิมชื่อ "มหาชัย" เป็นเมืองท่าที่มีความสำคัญตั้งแต่สมัยกรุงศรีอยุธยา ต่อมาเปลี่ยนชื่อเป็นสมุทรสาครในรัชกาลที่ 4 เมืองนี้มีบทบาทด้านการประมงและการค้าทางทะเลมายาวนาน

เศรษฐกิจ

สมุทรสาครเป็นศูนย์กลางอุตสาหกรรมประมง การแปรรูปสัตว์น้ํา รวมถึงโรงงานอุตสาหกรรมหลากหลายประเภท ตลาดมหาชัยเป็นตลาดอาหารทะเลที่มีชื่อเสียง

แกลเลอรี

แทนที่รูปภาพในโฟลเดอร์ images/ ด้วยรูปจริงก่อนอัปโหลด

วิธีนำขึ้น GitHub (สรุป)

ขั้นตอนสั้น ๆ เพื่ออัปโหลดโปรเจกต์นี้เป็น repository และใช้ GitHub Pages

  1. สร้างโฟลเดอร์โปรเจกต์ แล้ววางไฟล์นี้เป็น index.html และเพิ่มโฟลเดอร์ images/ ที่มีรูปจริง
  2. เปิดเทอร์มินัลแล้วรันคำสั่งต่อไปนี้:
git init git add . git commit -m "Initial commit: Samut Sakhon presentation" git branch -M main git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git git push -u origin main

จากนั้นไปที่ GitHub > Settings ของ repository เลือก Pages และตั้งเป็นสาขา main เพื่อเปิดใช้งาน GitHub Pages
